## Taking a screen shot from terminal.

### Install imagemagick
`$ sudo apt install imagemagick;`

### Take a shot of the whole screen
`$ import -window root image.png`

### Take a shot of an area
`$ import image.png`

## OR

### Install scrot
`$ sudo apt install scrot`

### Take a shot of the whole screen
`$ scrot image.png`

### Take a shot of an area
`$ scrot -s image.png`
